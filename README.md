# SLC Weather

A weather app displaying Salt Lake City weather forecasts using [Dark Sky](https://darksky.net/dev/) and [React](https://facebook.github.io/react/).

## Getting Started
```bash
git clone https://github.com/mshiloh/slc-weather.git
```
